Linear Quadratic Regulator (LQR) for Inverted Pendulum

This project applied a Linear Quadratic Regulator (LQR) to stabilize an inverted pendulum on a cart system. Using Newtonian physics, the nonlinear dynamics of a 2-DOF system (cart position and pole angle) were derived and linearized to form state-space equations.

Three simulation cases were run in MATLAB/Simulink to observe how different Q and R weightings in the cost function impacted system performance:

  • Case 1 (Baseline): Identity matrix Q and small R value. Smooth motion reaching the target in ~4s.
  • Case 2 (Fast Response): Lowered Q velocity weight. System reached 10m twice as fast with higher control effort.
  • Case 3 (High R): Large R value reduced control effort but failed to reach the final position in 10s.

The simulation demonstrated how LQR can be tuned to prioritize speed, accuracy, or energy efficiency by adjusting Q and R. The optimal gains were calculated using MATLAB's lqr() function, and step responses were evaluated in Simulink.

This project laid a foundation for advanced control topics like LQG and Kalman filtering. It highlights the power of model-based control design when system dynamics are well-understood.
