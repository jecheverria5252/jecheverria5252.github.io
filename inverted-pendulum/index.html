Inverted Pendulum Project

This project focused on simulating an inverted pendulum using classical control techniques in Simulink. The goal was to stabilize the system via a feedback controller and analyze performance under different disturbance inputs.

The project included:

  • Mathematical modeling of the pendulum system
  • Simulink-based system simulation
  • Controller design using pole placement and LQR
  • Performance analysis with step and impulse responses
