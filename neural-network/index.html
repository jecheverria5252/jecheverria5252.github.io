Sequence-to-Sequence Neural Network for Control Systems

This project explored the use of Sequence-to-Sequence (Seq2Seq) neural network architectures to model and predict control outputs in closed-loop temperature regulation systems.

Traditional PID controllers were compared to data-driven models trained on time-series temperature and control signal data. The neural networks learned to predict future control actions based on historical sequences of input-output data.

Highlights include:

  • Implemented encoder-decoder models in TensorFlow/Keras
  • Preprocessed time-series control system data for sequence learning
  • Trained models to improve temperature regulation stability
  • Evaluated model performance against traditional control strategies
